# UPLOAD DE CSV PROJECT

### Stack Used:

- Nodejs
- Multer
- EJS
- Express

### Como Iniciar o projeto:

- clone the project

```
bash
git clone 
```

- Depois de clonar o projeto no seu local favorito, abra-o com sua ide favorita
- Agora no terminal de sua ide use os seguintes comandos:
- `npm install`
- `npm start`
